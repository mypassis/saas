# Nebula - Next Gen SaaS Platform

A high-converting, modern SaaS landing page built with HTML and CSS. It features a sleek **Glassmorphism** design, responsive layout, and interactive elements to drive user engagement.

## ✨ Features

*   **Modern Glassmorphism UI**: Translucent backgrounds with blur effects for a premium look.
*   **Sticky Navbar**: Keeps navigation accessible while scrolling.
*   **Hero Section**: Eye-catching gradient text and call-to-action buttons.
*   **Feature Cards**: Interactive cards with smooth hover effects.
*   **Responsive Design**: Optimized for mobile, tablet, and desktop views.
*   **No External JS Frameworks**: Pure HTML/CSS for lightweight performance.

## 🚀 How to Run

1.  **Clone or Download** the project files.
2.  Ensure you have the following files in your directory:
    *   `index.html`
    *   `style.css`
3.  **Open** `index.html` in any modern web browser.

Alternatively, use a local development server (e.g., VS Code Live Server) for the best experience.

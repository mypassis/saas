# Minimalist Design System

## Aesthetics
Clean, minimalist, and editorial. Avoid clutter. Focus on typography and spacing.

## Colors
Strictly two-tone:
- Background: `#111111` (Matte Charcoal)
- Text/Accents: `#F2F2F2` (Off-White)
- Borders: `#333333`

## Typography
Use 'Inter' for all text.
- Headings: Bold, tight tracking (-1%).
- Body: Regular weight, generous line height (1.6).

## Layout
Full-width layouts with centered content containers.
- No rounded corners (border-radius: 0).
- Sharp edges.

## Interactions
Slow, elegant transitions (0.5s ease-out).
- Hover states should be subtle (opacity changes or slight translation).
- No glowing effects.
